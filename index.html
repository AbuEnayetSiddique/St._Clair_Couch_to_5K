Everyone Can Be a Runner!

Whether you've never run before, or haven't run for a while, everyone can become a runner with our simple St. Clair Couch to 5K Program (SCC25K). Couch to 5K is a system where you follow a proven training system that gradually, safely trains your body to run a 5K distance without injury. Browse this site to learn how and get started today!
